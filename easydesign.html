About Us

EasyDesign Solution provides engineering analysis, simulation and technical visualization services using advanced open-source software platforms.

We focus on affordable engineering solutions for bridges, underpasses, road structures and industrial applications.

Our objective is to deliver efficient simulation-driven designs using CFD and structural analysis technologies.

Open Source Software We Use

  • OpenFOAM – CFD Analysis
  • CalculiX – Structural Analysis
  • Code_Aster – Finite Element Analysis
  • Salome Platform – Pre and Post Processing
  • FreeCAD – CAD Modeling
  • Blender – Engineering Visualization

We use open-source engineering software for simulation, analysis and visualization services.
